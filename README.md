# MindNexus – Private AI Knowledge Brain

MindNexus is a local Retrieval-Augmented Generation (RAG) system that allows
users to interact with PDFs using chat and voice.

## Features
- Local LLM (Ollama – Llama 3.2)
- Multi-PDF Knowledge Libraries
- Admin/User Authentication
- Persistent Chat History
- Voice Interaction
- Privacy-Preserving AI

## Run
streamlit run app.py
